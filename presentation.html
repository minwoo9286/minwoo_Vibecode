Overview · 서비스 소개

풋살 동호회 운영의
모든 것을 웹 하나로

동호회 팀 모집, 대진표 작성, 경기 결과 기록까지 모두 수작업으로 처리하던 불편함을 해소합니다.
별도 앱 설치 없이, 회원가입만으로 바로 사용 가능한 풀스택 웹 서비스입니다.

🎯 해결하는 문제

  • 카카오톡으로 상대팀 수작업 섭외
  • 엑셀로 대진표 매번 직접 작성
  • 경기 결과 기록이 단톡방에서 소실
  • 시즌 종료 후 우승 기록 사라짐
  • 운영진 한 명에게 행정 부담 집중

✅ 우리의 해결책

  • 웹에서 팀 등록 후 토너먼트 참가 신청
  • 대진표 자동 생성 (싱글 엘리미네이션)
  • 경기 결과 온라인 입력 및 확정
  • 명예의 전당 — 역대 우승팀 영구 기록
  • 관리자 권한으로 토너먼트 자율 운영
Features · 주요 기능

6가지 핵심 기능

  • 01 이메일 인증 기반 회원가입 / 로그인
  • 02 팀 등록 — 이름, 지역(시도·시군구), 대표자명
  • 03 토너먼트 개최 — 이름, 지역, 최대 팀 수, 설명 설정
  • 04 토너먼트 참가 신청 — 드롭다운에서 팀 선택 후 신청
  • 05 자동 대진표 생성 + 경기 결과 입력 (관리자 전용)
  • 06 명예의 전당 — 완료된 토너먼트 우승팀 기록 보존
Team · 팀 관리

팀 등록 & 목록 조회

사용자는 팀 이름, 소속 지역(시도 / 시군구), 대표자 이름을 입력해 팀을 등록합니다.
등록된 팀은 목록에서 바로 확인 가능하며, 하나의 팀으로 여러 토너먼트에 참가할 수 있습니다.

⚽ 팀 등록 정보

  • 팀 이름 (필수)
  • 지역 — 시도 선택 (서울, 경기, 부산 …)
  • 지역 — 시군구 선택 (선택사항)
  • 대표자 이름 (선택사항)

📋 팀 목록 페이지

  • 전체 등록 팀 수 표시
  • 팀명 + 지역 + 대표자 한눈에 확인
  • 등록 순 최신 정렬
  • 팀 등록 버튼으로 바로 이동
